Get the app View source

What you get

Study path

Hand rankings, position, ranges, pot odds, board textures, c-bets, combinatorics, blockers, and exploits — with quizzes you can practice endlessly.

Play sandbox

6-max table against AI bots. Bot types stay fixed per session so you can learn their tendencies over many hands.

Read opponents

Guess or peek player type, hole cards, and opening range on the 13×13 matrix. Score your accuracy.

Math & coaching

Show Math for outs, equity, and pot odds anytime. EV Coach flags -EV calls with clear explanations.

Sessions

Chips carry over between hands. End a session for stats and download hand histories in PokerStars format.

Offline first

Runs entirely on your machine. macOS and Linux builds available.

Install

Download the latest release for your platform from GitHub:

  • macOS — open the .dmg, drag All-In to Applications
  • Ubuntu / Debian — install the .deb package
  • Linux — run the .AppImage (chmod +x first)

Build from source if you prefer.
